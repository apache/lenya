Cocoon

Apache Cocoon is an XML publishing framework that raises the usage of XML and XSLTtechnologies for server applications to a new level. Designed for performance andscalability around pipelined SAX processing, Cocoon offers a flexible environmentbased on a separation of concerns between content, logic, and style. To top this alloff, Cocoons centralized configuration system and sophisticated caching help you tocreate, deploy, and maintain rock-solid XML server applications.

http://cocoon.apache.org

Slide

The Slide project main module is a Content Management and Integration System, whichcan be seen as a low-level content management framework. Conceptually, it providesa hierarchical organization of binary content which can be stored into arbitrary,heterogenous, distributed data stores. In addition, Slide integrates security, locking,versioning, as well as many other services.

http://jakarta.apache.org/slide/
