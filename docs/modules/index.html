Apache Lenya Modules Documentation

Warning
Some of Apache Lenya modules are still under heavy development. Please consult the module documentation about current state of development.

Introduction

Modules are packages providing a certain set of resources or functionality. The aim of this section in the documentation is to group the information needed to work with a certain module into one entry point.

To learn how modules are implemented, refer to the general modules documentation.

Modules

blog
Resource types and Java classes used by the blog publication. They are not yet ready to be used separately.
bxe
Integration of the Bitflux WYSIWYG editor.
cforms
A resource type which can be edited using Cocoon Forms.
collection
A resource type to store collections of documents.
contactform
A simple contact form. This is a usage example for the usecasedocument module.
development
Rresources to simplify development of Lenya applications.
editors
Form-based editors.
fckeditor
Integration of the FCK WYSIWYG editor.
kupu
Integration of the Kupu WYSIWYG editor.
languageselector
XHTML widget to switch languages on a website.
lenyadoc
Support for the lenyadoc:/ protocol.
linkcheck
Link checking functionality.
links
A resource type to maintain lists of links.
lucene
Integration of the Lucene search engine.
migration
Resources to simplify the migration between different Lenya versions.
neutron
Server-side implementation of the Neutron protocol to support browser-based editors.
news
A resource type to display lists of news messages, can be used as a simple weblog.
notification
Support for user notification and inboxes.
opendocument
OpenDocument (ODF) resource type.
profiling
Resources to simplify profiling (performance optimization).
resource
The "resource" resource type, it allows to store arbitrary media documents.
simplesite
A simple site manager based on collections.
sitetree
Tree-based site manager.
sourcerepository
Repository implementation based on Avalon sources.
svg
Resources to support the rendering of SVG-based images, e.g. for boxes with rounded corners.
tinymce
Integration of the TinyMCE WYSIWYG editor.
usecasedocument
Resource type which allows to include a usecase as a document in the site structure.
webdav
WebDAV support.
xhtml
Resource type which allows to store XHTML documents.
xopus
Integration of the Xopus WYSIWYG editor.
