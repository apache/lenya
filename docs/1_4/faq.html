Frequently Asked Questions

Questions

1. Getting Started and Building Lenya

1.1. How to extend the i18n catalog for Lenya 1.4?

^

In the default publication you can find pubs/default/lenya/resources/i18n. That are the files you need to edit if you wish to override it on a publication base.

However you can as well extend the catalog with modules, that you can reuse in different publications. You need to add at least modules/{yourModule}/lenya/resources/i18n/cmsui.xml.

1.2. Is it possible to override in BXE *only* a css style definition?

^

Yes, this is possible.
Just add and edit it like you need it (have a look in the default pub): {pubid}/resources/misc/bxe/{ressourcetype}-bxe.css
