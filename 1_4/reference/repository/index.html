The Lenya Content Repository

Note
This document is a draft. It does not reflect the current implementation, but will evolve alongside the code.

Introduction

The Lenya content repository is based on the Java Content Repository API.

Repository Layout

The JCR repository used by Lenya comprises a set of workspaces, each of which represent an area of the document structure. For each area that is declared by a publication, a repository workspace is created.

Publications are orthogonal to workspaces. Each publication can declare its own set of areas. Thus, the set of workspaces in the Lenya repository derives from the union of all areas declared by publications.
