Subversion for the public

Start the shell of your choice and enter: 

svn co http://svn.apache.org/repos/asf/lenya/trunk lenya-1.4.x

Subversion for committers

Start the shell of your choice and enter: 

svn co https://svn.apache.org/repos/asf/lenya/trunk lenya-1.4.x
ssh -l username svn.apache.org
svnpasswd username
exit

View Subversion

http://svn.apache.org/viewcvs.cgi/lenya/#dirlist

Subversion Tips

Installing the SVN client with SSL enabled on Debian Woody: 

apt-get install openssl
apt-get install libssl-dev (Important Header Files)

./configure --with-ssl
make
make install (su)
svn --version
