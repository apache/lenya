It seems that Kupu is not built properly

NOTE: If you are using a SVN checkout of Lenya, you need to run make in the Kupu root directory or ./build.sh kupu on Linux or build.bat kupu on Windows in the Apache Lenya source directory. In the case of Tomcat as servlet container one needs to run additionally ./build.sh install resp. build.bat install. Make sure that xsltproc is installed on your system.

Lenya binary releases ship with Kupu already preconfigured.

Many thanks to the Kupu community for making this great browser based XHTML WYSIWYG editor available.
