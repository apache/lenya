It seems that Epoz is not installed on your server
Lenya is expecting an epoz directory within your webapps/ROOT directory.

Please download and install Epoz as follows

Download the head version of Epoz from http://www.codespeak.net/svn/epoz/trunk/epoz/ via Subversion
(svn co http://www.codespeak.net/svn/epoz/trunk/epoz/)
and copy or move the epoz directory to webapps/ROOT

No further setup is required.

Note that if you use Jetty, you will have to adjust paths because Lenya runs in the ROOT context.

Many thanks to the Epoz community for making this great tool available.
