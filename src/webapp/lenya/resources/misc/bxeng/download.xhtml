It seems that BXENG is not installed on your server

Lenya is expecting a bxeng directory within your webapps ROOT directory.

Please download and install BXENG as follows

cd $TOMCAT/webapps/ROOT
cvs -d :pserver:anonymous@cvs.bitflux.ch:/var/lib/cvs/editor co bxeng

(No password required)

That's it :-)

Many thanks to the Bitflux Editor community for making this great tool available.
