NOTE: If you are using a SVN checkout of Lenya, you need to run make in the Kupu root directory or ./build.sh kupu on Linux or build.bat kupu on Windows in the Apache Lenya source directory. Make sure that xsltproc is installed on your system.

Lenya releases ship with Kupu already preconfigured.
