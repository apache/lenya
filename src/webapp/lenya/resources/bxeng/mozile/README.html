mozile 0.5 with eDOM Installation

Three simple steps to use ...

  1. unzip mozile-install.zip somewhere on your web site
  2. put a reference to "mozileLoader.js" in the head element of any file that you want to edit: 
    script src=".../mozileLoader.js" type="text/javascript"
  3. Change the method mozileSave in mozileModify.js to persist Mozile changes appropriately for your application.

Note: this is safe in IE and Safari because loader only activates mozile in Geiko-based browsers!
