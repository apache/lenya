Important

You need Mozilla, otherwise it won't run.
We recommend Mozilla 1.6 or Mozilla Firefox 1.0 (our current testing platform) . Mozilla 1.7 and Firebird 0.9 do not save correctly, due to a bug in the XML Serializer of Mozilla.

  • Hit F7 to get a cursor
  • Save doesn't work in the demo, since this is normally handled on the server by something like a CMS. Here it's just static file calls.

Examples

  • Bitflux Editor Website with inline loading and XHTML Elements (with image insert support)
  • Bitflux Editor Website with inline loading
  • University of Zurich (has XHTML Schema support, quite complex)
  • Echo/Atom (needs a nicer CSS and a fixed RelaxNG)
