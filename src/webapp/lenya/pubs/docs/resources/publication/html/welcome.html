Documentation Wyona


Wyona Basics:

Here you will find a documentation about the basic concepts and the implementation of Wyona.


User's Guide:

Here you will find a documentation about the handling of Wyona as a User. Users typically are concerned with problems such as creating, editing and publishing articles.


Administrator's Guide:

Here you will find a documentation about the administration of Wyona. Administrators typically have to deal with problems arising from maintenance concerning User's and their profiles or the site, publication and page description.


Integrator's Guide:

Here you will find a documentation about the integration of Wyona. Integrators not only have to manage the installation, but are also responsible for the design ond layout of the publication.


Developer's Guide:

Here you will find a documentation about the development of Wyona. Developers are concerned with the improvement of the source code of Wyona.


Javadoc:

Here you will find a documentation about the Java classes used in Wyona and Cocoon.
