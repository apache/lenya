The Wyona-CMS Scratchpad

These documents are for keeping track of the road ahead, as well as providing a discussion base for issues encountered during the refactoring of the Wyona XPS (as used for NZZ Online) to the Wyona CMS, which is based on open standards, especially Cocoon, an XML publishing framework being developped by the Apache Software Foundation. They serve as to-do lists and path descriptions for the developpers, as well as a digest of the Wyona mailing list). Of course, most of what is on this scratchpad is still open to discussion, and everybody is invited to contribute.
