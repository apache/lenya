Customizing

Layout

Frontpage (Live)

In order to change the layout, you have to edit the following stylesheets:
stylesheets/publication/Page/Front/Live/main.xsl
stylesheets/publication/Page/root.xsl

There are currently two layouts:
stylesheets/publication/Page/page_wyona.xsl
stylesheets/publication/Page/Front/body_wyona.xsl
and
stylesheets/publication/Page/page_q42.xsl
stylesheets/publication/Page/Front/body_q42.xsl

Navigation (Layout: Wyona)

At the moment you have to edit the following XSLT in order to modify the global navigation on the left hand side:

stylesheets/publication/Page/page_wyona.xsl

Of course the navigation would actually belong into a common XML, but who is going to change that?!
