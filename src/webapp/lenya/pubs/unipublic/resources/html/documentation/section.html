New Year

When the new Year beginn, we should

  • create new directories for this year: for the articles, for all sections and channels. Exp:

    • .../pubs/unipublic/docs/publication/authoring/magazin/geist/articles/2003
    • .../pubs/unipublic/docs/publication/authoring/magazin/gesundheit//articles/2003
    • .../pubs/unipublic/docs/publication/authoring/magazin/recht/articles/2003
    • .../pubs/unipublic/docs/publication/authoring/magazin/.../articles/2003
    • .../pubs/unipublic/docs/publication/authoring/campus/.../articles/2003
    • .../pubs/unipublic/docs/publication/authoring/.../.../articles/2003

  • put the corresponding branches in the tree: .../pubs/unipublic/docs/publication/authoring/tree.xml:

    <?xml version="1.0"?/>
    <tree>
     <branch relURI="" doctype="Front" menuName="Home">
       <branch relURI="magazin" doctype="Channel" menuName="Magazin">
         <branch relURI="articles" doctype="Articles" menuName="Articles">
          <branch relURI="2002" doctype="Year" menuName="2002"/>
          <branch relURI="2003" doctype="Year" menuName="2003"/>
         </branch>
        </branch>
        <branch relURI="gesundheit" doctype="Section" menuName="Gesundheit">
         <branch relURI="articles" doctype="Articles" menuName="Articles">
          <branch relURI="2002" doctype="Year" menuName="2002"/>
          <branch relURI="2003" doctype="Year" menuName="2003"/>
         </branch>
        </branch>
        ....
       </branch>
       <branch relURI="channel" doctype="Channel" menuName="Campus"/>
        <branch relURI="section" doctype="Section" menuName="Uni-News">
         <branch relURI="articles" doctype="Articles" menuName="Articles">
          <branch relURI="2002" doctype="Year" menuName="2002"/>
          <branch relURI="2003" doctype="Year" menuName="2003"/>
         </branch>
        </branch>
        ....
       </branch>
       ...
      </branch>
    </tree>

  • change the hardcoded redirect in the sitemaps: .../pubs/unipublic/authoring-sitemap.xmap and .../pubs/unipublic/live-sitemap.xmap:

    ...
     <map:match pattern="*/*/">
       <map:redirect-to uri="2003/index.html" session="true"/>
      </map:match>
      <map:match pattern="*/*/index.html">
       <map:redirect-to uri="2003/index.html" session="true"/>
      </map:match>
    ...

  • put the old year in the hardcoded year's navigation in the section stylesheet: .../pubs/unipublic/stylesheets/publication/Page/Section/page.xsl:

    <b><a href="../{$year}/"><xsl:value-of select="$year"/></a></b> | <a href="../2002/">2002</a> | <a href="http://www.unipublic.unizh.ch/{$channel}/{$section}/2001/">2001</a> | <a href="http://www.unipublic.unizh.ch/{$channel}/{$section}/2000/">2000</a> | <a href="http://www.unipublic.unizh.ch/{$channel}/{$section}/1999/">1999</a>
