Prerequisites

The following list describes the setup that is tested and recommended. Several items, e.g. the servlet container, may be replaced by other products.

  • Java 2 Platform, Standard Edition (required: J2SE 1.4.1 or newer)
    http://java.sun.com/j2se/1.4.2/download.html

  • Apache Tomcat for JDK 1.4 (recommended: 4.1.29)
    Tomcat release builds
    Pick one of the jakarta-tomcat-4.1.24-LE-jdk14 files!

  • Apache Ant (required: 1.6.1 or newer)
    Ant main distribution directory

  • Apache Cocoon (required: Cocoon 2.1.4)
    Cocoon download page

  • Endorsed Libraries (recommended: Xalan 2.5.0 Xerces 2.5.0)
